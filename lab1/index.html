  • ¿Cuáles son las partes de un URL?

  • ¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?

    GET: recupera un recurso, HEAD: al igual que get recupera un recurso, pero sin el cuerpo de la respuesta, POST: envia una entidad a un recurso en especifico, PUT: crea un nuevo elemento o reemplaza el elemento del destino, PATH:Aplica modificaciones a un recurso, DELETE: borra un recurso

  • ¿Qué método HTTP se debe utilizar al enviar un formulario HTML, por ejemplo cuando ingresas tu usuario y contraseña en algún sitio? ¿Por qué?

    Para el caso de enviar formularios por medio de html se debe utilizar el metodo POST por que por medio de este metodo se envian los datos al servidor

  • ¿Qué componentes de estructura y estilo tiene una tabla?

    Los componentes de la tabla son: CAPTION: da un título para la tabla, COLGROUP: agrupa columnas., COL: aplica atributos y estilos a una o más columnas, TBODY: representa a un elemento de la tabla., THEAD: Es la sección de encabezado de la tabla., TFOOT: Es la sección de pie de la tabla.

  • Sobre el ciclo de vida y desarrollo de los sistemas de información: ¿Cuál es el ciclo de vida de los sistemas de información?

    El ciclo de vida de un sistema de informacion es:

  • Fase de planificacion: Se prepara el disenio y se planifica
  • Fase de analisis: Analisis y actualizacion de requisitos
  • Fase de disenio: Se determina como el sistema cumplira los requisitos
  • Fase de desarrollo: Se desarrolla el sistema, en esta fase es donde mas cambios ocurren
  • Fase de integracion y pruebas: Se corrobora que el sistema cumpla con los requisitos solicitados
  • Fase de implementacion: Se mueve a produccion, creando documentacion y capacitando a usuarios.
